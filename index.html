Projects

Job Hunt Organizer (Current)

Job Hunt Organizer is an app that aims to change the job hunt by promoting connectivity between people. Other features include automatically compiling information for companies of interest and prioritizing tasks based on recency and urgency.

Maps with Friends (March, July)

Maps with Friends is a Sinatra app leveraging the Google Maps API to allow users to add markers to the map with info windows displaying the name and location of their friends.

Hummingbird (February, April)

Hummingbird is a web app that allows users to schedule text messages to be sent at a more optimal time in the future. Hummingbird consists of an Ionic/Angular client and a decoupled Rails API. Hummingbird was built in seven days by myself, Jeremy McIntyre, William Tsang, and Rex Salisbury.
